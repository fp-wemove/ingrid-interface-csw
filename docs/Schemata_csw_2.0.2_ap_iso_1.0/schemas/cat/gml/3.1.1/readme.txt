These are preliminary versions of the GML 3.1.1 bugfix release. 

This release is being developed with minimal corrections relative to GML 3.1.0 to ensure successful validation by the most common processing tools, in particular Xerces-J and .Net. 
The intention is for this schema to be submitted to Open Geospatial Consortium, and expected to be made available from the official OGC repository at the address

http://schemas.opengis.net/gml/3.1.1/
